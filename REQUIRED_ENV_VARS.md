# Required Environment Variables

All variables must be set in Vercel → Project Settings → Environment Variables.

---

## Frontend (Vite build-time, prefix `VITE_`)

These are bundled into the client JS at build time. Safe to be public.

| Variable | Description |
|---|---|
| `VITE_RAZORPAY_KEY_ID` | Razorpay publishable key (starts with `rzp_`) |
| `VITE_RAZORPAY_PLAN_INDIA_MONTHLY` | Razorpay plan ID for India monthly subscription |
| `VITE_RAZORPAY_PLAN_INDIA_ANNUAL` | Razorpay plan ID for India annual subscription |
| `VITE_RAZORPAY_PLAN_GLOBAL_MONTHLY` | Razorpay plan ID for global monthly subscription |
| `VITE_RAZORPAY_PLAN_GLOBAL_ANNUAL` | Razorpay plan ID for global annual subscription |
| `VITE_BLOG_GENERATION_SECRET` | Secret for blog admin generation endpoint |

> Note: Supabase URL and anon key are hardcoded in `src/integrations/supabase/client.ts` (auto-generated by Supabase — anon key is safe to be public).

---

## API / Server-side (Vercel serverless functions)

These are **never** exposed to the browser.

| Variable | Description |
|---|---|
| `RESEND_API_KEY` | Resend API key for transactional emails (`re_...`) |
| `SUPABASE_URL` | Supabase project URL (`https://<ref>.supabase.co`) |
| `SUPABASE_SERVICE_ROLE_KEY` | Supabase service role key — **never expose to browser** |
| `RAZORPAY_WEBHOOK_SECRET` | Secret for verifying Razorpay webhook signatures |
| `ANTHROPIC_API_KEY` | Anthropic API key for AI Longevity Coach (`sk-ant-...`) |
| `CRON_SECRET` | Random secret Vercel attaches to cron requests (`Authorization: Bearer <CRON_SECRET>`) |

---

## Vercel Automatic (set by platform — do NOT override)

| Variable | Description |
|---|---|
| `VERCEL_URL` | Preview deployment URL (auto-set) |
| `VERCEL_PROJECT_PRODUCTION_URL` | Production domain (auto-set) |

---

## Setup checklist

- [ ] All `VITE_` vars set for Production, Preview, and Development environments
- [ ] `RESEND_API_KEY` added and domain `bornclock.com` verified in Resend dashboard
- [ ] `SUPABASE_SERVICE_ROLE_KEY` set (copy from Supabase → Project Settings → API)
- [ ] `RAZORPAY_WEBHOOK_SECRET` matches the secret set in Razorpay Dashboard → Webhooks
- [ ] `ANTHROPIC_API_KEY` active with sufficient credits
- [ ] `CRON_SECRET` set to a random 32+ char string (used by Vercel cron and admin manual trigger)
